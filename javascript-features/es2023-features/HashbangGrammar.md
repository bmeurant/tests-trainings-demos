# Hashbang Grammar

1. add execution permissions to the `hashbangGrammar.js` file:

   ```bash
   chmod +x features/hashbangGrammar.js
   ```
   
2. run directly the `hashbangGrammar.js` file:

   ```bash
   ./features/hashbangGrammar.js
   ```
   
3. run it with node:

   ```bash
   node features/hashbangGrammar.js
   ```
